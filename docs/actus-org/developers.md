# Developers

## Registration

All parties who wish to access the ACTUS GitHub open-source repository are invited to register on the ACTUS website. This registration requirement enables ACTUS to comply with regulatory requirements.

## Reference Implementation

The ACTUS Users Association develops, maintains, and releases a reference implementation in Java through GitHub under the ACTUS Core License.

The Java reference implementation includes the actus-core library, which is the core library implementing the Contract Types. The actus libraries constitute the algorithmic standards as Java libraries.

## ACTUS Core License

The reference implementation is distributed as FOSS software under the ACTUS Core License. The license terms include restrictions on how modified software can be redistributed, in order to guarantee the integrity of the implemented contract logic with the ACTUS Technical Specification.

The ACTUS Core License is maintained in the GitHub repository `actusfrf/actus-core-license`.

## GitHub Organization

The ACTUS Financial Research Foundation maintains a GitHub organization at `github.com/actusfrf`. Repositories include:

- `actus-core` — the core Java library implementing the Contract Types
- `actus-techspecs` — the ACTUS Technical Specifications in LaTeX
- `actus-dictionary` — the ACTUS data dictionary and generation scripts
- `actus-core-license` — the ACTUS Core License
- `actus-service` — the ACTUS service
- `actus-webapp` — the ACTUS Demo App web application
