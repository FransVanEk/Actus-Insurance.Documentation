# Resources

## Community Edition

The ACTUS Community Edition download and related demo apps make it easy to see how ACTUS can be used to incorporate forward-looking cash flow analysis, risk management, or digital asset management capabilities in a financial business.

## Demo Application

The ACTUS Demo App is a lightweight web application showcasing the ACTUS standards. The demo can be accessed at `demo.actusfrf.org`. Opening that address in a browser brings up a list of contract types that can be explored interactively.

The source code of the ACTUS Demo App is available in the GitHub repository `actusfrf/actus-webapp`.

## Quick Start

The ACTUS Quick Start provides a step-by-step guide to quickly set up and use the ACTUS framework using Docker. The Quick Start page provides an index to helpful documentation and instructions regarding where to find and access resources, demos, and downloadable ACTUS packages.

The Quick Start is available at `documentation.actusfrf.org/docs/quickstart`.

## Documentation

Comprehensive documentation is available at `documentation.actusfrf.org`. The documentation includes guides on modeling risk factors, simulating contracts, and working with risk databases, as well as hands-on demonstrations and practical examples.
