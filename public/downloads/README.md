# Downloads

Place downloadable files here (PDF, PowerPoint, video, Excel, etc.).

Files in this folder are served at `/downloads/<filename>` and referenced
from `config/resources.json`.

## Naming convention

Use kebab-case filenames that match the `url` field in `resources.json`:

```
actus-overview.pdf
hackathon-slides.pptx
demo.mp4
pam-test-data.xlsx
```
